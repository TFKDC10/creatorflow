# CreatorFlow

A clean, minimal productivity web app for creators with Idea Inbox, Task Board, Build Tracker, and Ship Log.

## Framework

Vanilla JavaScript with Tailwind CSS (CDN)

## Preview

Configured to run on `0.0.0.0:8080` for k8s HTTPRoute access.

## Theme

Clean Light - Modern light mode with white/gray backgrounds, soft shadows, and subtle borders for a premium, minimal feel.

## Features

- **Dashboard**: Overview of all your productivity metrics
- **Idea Inbox**: Capture and manage creative ideas
- **Task Board**: Kanban-style task management (To Do, In Progress, Done)
- **Build Tracker**: Track progress on ongoing projects
- **Ship Log**: Celebrate your launches and releases

All data is stored locally in your browser's localStorage.
